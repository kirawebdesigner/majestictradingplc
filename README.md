# Majestic Trading PLC Homepage Concept

Premium Vite/React homepage concept for Majestic Trading PLC, based on the Strux Framer template direction and rebranded around Majestic's construction, interior design, branding, digital services, system development, and business solutions.

## Local Development

```bash
npm install
npm run dev -- --port 5173
```

Local URL:

```text
http://127.0.0.1:5173/
```

## Production Build

```bash
npm run build
```

Output directory:

```text
dist
```

## Netlify Deployment

This repo is ready for Netlify with `netlify.toml`.

- Build command: `npm run build`
- Publish directory: `dist`
- Node version: `22`
- SPA fallback: enabled
- Basic security/cache headers: enabled

Preview deploy:

```bash
npx netlify deploy
```

Production deploy:

```bash
npx netlify deploy --prod
```

## Content Assumptions

Some project names, metrics, and system-development/business-solution details are concept placeholders until Majestic provides verified case studies and internal service details.
